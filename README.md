## Directory structure

This repo is structured with [flake-parts](https://flake.parts/getting-started.html) and [den](https://github.com/vic/den) for an aspect-oriented approach to Dendritic configurations.

```
nixpedition/
├── modules/
│   ├── hosts/             → Host configurations
│   ├── lib.nix            → Custom "batteries"
│   └── defaults.nix       → Defaults for all machines
├── nix/                   → Flake configurations
├── flake.nix              → Generated by vic/flake-file module
└── shell.nix              → nix-shell file generated by `mightyiam/files` module
```

## Machines

> [!TIP]
> All machines defined in this configuration are named using characters from the game [Clair Obscur: Expedition 33](https://store.steampowered.com/app/1903340/Clair_Obscur_Expedition_33/).

Here is a list of all my machines running NixOS:

- [**Maelle**](/modules/hosts/maelle) - My main workstation where I do most of my work and play games.
- [**Sciel**](/modules/hosts/sciel) - My [Framework 16](https://frame.work/pl/en/laptop16) laptop for working on the go and ~~chilling in the Starbucks~~ (_I forgot that I don't go outside..._).
- [**Esquie**](/modules/hosts/esquie) - HomeLab server that hosts my private, self‑hosted applications.
- [**Francois**](/modules/hosts/francois) - VPS for publicly available services.

Each machine can communicate with each other using [Tailscale](https://tailscale.com/).
